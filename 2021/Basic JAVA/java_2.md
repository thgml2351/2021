## 문제 2

### 출력될 값을 예상하기

```
int i = 7;
int j = 3;
int h = 5;

System.out.println(i>j && i<h);
System.out.println(i>j || i>h);

int result = h != 5 ? 1 : 2;
System.out.println(result);
```
